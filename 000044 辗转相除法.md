# The Greatest Common Divisor

## Code

```C
#include <stdio.h>
int main() {
	int a, b, c, num1, num2;
	scanf("%d %d", &a, &b);
	num1 = a;
	num2 = b;
	while( b != 0 ) {
		c = a % b;
		a = b;
		b = c;
	}
	printf("The greatest common divisor of %d and %d is %d.", num1, num2, a);
	return 0;
}
```
