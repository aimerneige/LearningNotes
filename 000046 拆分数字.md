# Split A Number

## Code

```C
#include <stdio.h>

int main(void)
{
	int num, a = 10;

	printf("Please input a number.\n");
	scanf_s("%d", &num);

	for (a = 10; (num / a) >= 10; a *= 10) {}
	printf("%d  ", (num / a));
	do {
		num = num % a;
		a = a / 10;
		printf("%d  ", (num / a));
	} while (a != 1);

	return 0;
}
```
