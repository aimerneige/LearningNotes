# Minimum Number

## Code

```C
#include <stdio.h>
int main(void)
{
    int n, num, min;
    scanf("%d", &n);
    scanf("%d", &num);
    min = num;
    for (int i = 1; i < n; i++) {
        scanf("%d", &num);
        if (min > num) {
            min = num;
        }
    }
    printf("%d", min);
    return 0;
}
```
