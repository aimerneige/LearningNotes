# 八进制和十六进制



在c语言中，以**十进制显示数字**使用**%d,**

​                     以**八进制显示数字，**使用**%o**；

​                      以**十六进制显示数字,**使用**%x**；

 **注意：**如果要在八进制和十六进制的值前显示**0（八进制标志**）和**0x（十六进制标志）**前缀，

要分别在转换说明中加入#：

​                   比如：要在八进制的值前显示0，则在输出语句中写**”a=%#o“，a；**

​                             要在十六进制前显示0x，则在输出语句中写**“a=%#x”，a；**
