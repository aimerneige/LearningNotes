## 协作说明

### CCode

1. 在`CCode`目录下请按照模板书写内容
2. 不满足模板的内容不会被合并
3. 新建文件按照`序号 说明`来创建如： `000027 求和函数.md`
4. 编辑中的内容在序号后加下划线来区分如： `000027_ 求和函数.md`
5. 其他内容可以创建新的文件夹
